# Wayang Elektronik - Automation Testing

## About
This repository contains automation test scripts for the **Wayang Elektronik Sales & Inventory System**.  
The tests are created using **Katalon Studio** to validate basic features such as login, product management, transactions, and reports.

This project is used as part of my learning and junior QA portfolio.


## Tools
- Katalon Studio
- Selenium (via Katalon)
- Groovy
- Laravel (Application Under Test)
- Git & GitHub

## How to Run
1. Install **Katalon Studio**  
2. Clone this repository:
   ```bash
   git clone https://github.com/dikadzikrii/wayang-elektronik-automation.git
3. Open the project in Katalon Studio
4. Run any Test Suite
