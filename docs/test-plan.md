# Test Plan  
Wayang Elektronik - Automation Testing

## 1. Tujuan
Dokumen ini menjelaskan rencana pengujian sederhana untuk memastikan fitur utama pada sistem Wayang Elektronik berjalan dengan baik.

---

## 2. Fitur yang Diuji
- Login & Logout
- Manajemen Produk (Tambah, Edit, Hapus, Lihat)
- Transaksi Penjualan
- Perubahan Stok
- Laporan Penjualan

---

## 3. Metode Pengujian
- Manual testing untuk validasi awal fitur
- Automation testing menggunakan Katalon Studio untuk regresi dasar
- Pengujian dilakukan dengan metode Black Box

---

## 4. Lingkungan Pengujian
- OS: Windows
- Browser: Google Chrome
- Aplikasi: Web (Laravel)
- Tools: Katalon Studio

---

## 5. Kriteria Sukses
Pengujian dianggap berhasil jika:
- Fitur utama dapat dijalankan tanpa error besar
- Tidak ada bug kritis pada proses login dan transaksi
- Hasil pengujian dicatat dan disimpan

---

## 6. Catatan
Rencana pengujian ini dibuat sebagai bagian dari proses belajar dan portfolio junior QA.
